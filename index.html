(Gribouille): Un logiciel de dessin en Java et Java FX

Gribouille est un logiciel de dessin en Java et Java FX. Il permet de dessiner des formes géométriques, de les colorier et de les déplacer. Il est conçu pour être simple à utiliser et accessible à tous.

(qr-gen.xyz): WebApp générateur de QRCode

Un modeste projet de création d'une WebApp de génération et personalisation de QRCode. Ce projet m'a notamment permis d'apprendre le HTML, le CSS, le JavaScript et maintenant le PHP et l'utilisation d'une base de données SQL.

(pyqt6-pdf-qrcode): Logiciel de présentation à base de QRCode et de joystick (Adapté aux handicapés)

Ce projet a été réalisé par des élèves de Terminale durant les heures consacrées à l'enseignement de la spécialité numérique et sciences informatiques (NSI), il constitue leur projet de l'année de Terminale.
